# Diwali Sales Data Analysis

## Project Overview
This project focuses on analyzing Diwali sales data to understand customer purchasing behavior, popular product categories, and sales trends during the festive season. The goal of this analysis is to extract meaningful insights that can help businesses improve marketing strategies and increase sales.

## Dataset
The dataset contains customer demographic information such as age, gender, occupation, city category, product category, and purchase amount.

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Steps
1. Data Cleaning
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Extracting Business Insights

## Key Analysis
- Gender-wise purchase analysis
- Age group purchasing behavior
- Top product categories during Diwali
- City category vs purchase amount
- Occupation-wise spending patterns

## Key Insights
- Male customers contributed more to total sales compared to female customers.
- The 26–35 age group had the highest purchase rate.
- Certain product categories generated the majority of revenue.
- Customers from Tier-1 cities showed higher purchasing power.
